# Contributing

感谢你愿意改进小猫伸展提醒器。这个项目是一个纯前端静态网页，适合提交轻量、清晰、容易验证的改动。

## 本地运行

```bash
npm run preview
```

然后打开：

```text
http://localhost:4173/
```

如果没有使用 npm，也可以直接运行：

```bash
python3 -m http.server 4173
```

## 适合贡献的方向

- 修复计时、通知、语音播放等交互问题
- 改进移动端或桌面端布局
- 优化无障碍访问，例如按钮文案、键盘操作和对比度
- 增加新的提醒文案或提醒音频
- 改进 README、部署说明或使用说明

## 提交建议

1. Fork 这个仓库。
2. 创建一个描述清楚的分支，例如 `fix-notification-permission`。
3. 保持改动聚焦，一次提交解决一个明确问题。
4. 提交 Pull Request，并说明你改了什么、怎么验证。

## 代码风格

- 不引入构建工具，除非它能明显简化项目维护。
- 保持纯静态部署可用。
- 不提交 `.DS_Store`、压缩包、临时文件或本地环境文件。
- 新增音频或图片素材时，请确认你拥有使用和再分发的权利。

## 反馈问题

提交 Issue 时，最好包含：

- 使用的浏览器和系统
- 复现步骤
- 期望结果
- 实际结果
- 如果和通知或语音有关，请说明浏览器权限状态
